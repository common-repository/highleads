Automatically embed the Highleads chatbot on your WordPress site for enhanced lead generation and customer support.

== Description ==

Highleads for WordPress allows you to seamlessly integrate your Highleads chatbot into your WordPress website. This plugin provides an easy way to deploy your chatbot for improved customer support, lead generation, and user engagement.

Key features:
* Simple setup process
* Customizable chatbot placement
* Seamless integration with your existing WordPress theme
* Improved user engagement and support capabilities

For more information about Highleads, please visit our [official website](https://www.highleads.co/).

== Frequently Asked Questions ==

= How do I get my Chatbot ID? =

1. Log in to your [Highleads account](https://console.highleads.co/)
2. Navigate to the chatbot settings tab
3. Copy your Chatbot ID from the designated field

= Is Highleads free to use? =

Highleads offers a free trial, allowing you to explore our features without any initial cost. For more information about our pricing plans after the trial period, please visit our [pricing page](https://www.highleads.co/pricing).

= Where can I find documentation? =

For detailed documentation and support, please [contact us](https://www.highleads.co/contact).

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/highleads` directory, or install the plugin through the WordPress plugins screen directly
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Highleads screen to configure the plugin
4. Enter your Chatbot ID in the provided field
